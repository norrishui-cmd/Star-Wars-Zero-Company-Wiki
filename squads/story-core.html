How to Play It

Open with scouting or cover setup, let the squad's safest role reveal enemy positions, then spend high-value actions only after a kill, objective step, or defensive reset is available.

Swap Rules

If the squad lacks sustain, reduce exposure. If it lacks burst, add a Sharpshooter or Gunslinger. If it lacks control, add Scout, Jedi, or suppression tools before hard missions.
